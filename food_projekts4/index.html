Sistema

Sistēma ir izstrādāta, lai palīdzētu lietotājiem aprēķināt makroelementus (olbaltumvielas, taukus, ogļhidrātus) un kalorijas, ko viņi patērē, ēdot dažādus pārtikas produktus.Sistēma sastāv no 9 galvenajiem failiem:

AppTest.java

This is the main application file. It provides a menu-driven interface for users to interact with the food library. Users can add, view, edit, delete, sort, and save food items through this file.

Food100g.java

This file defines the `Food100g` class, which represents the nutritional values of food per 100 grams. It includes properties like protein, fat, carbs, and calories.

Food.java

This file extended by the `Food100g` class to include the quantity of food consumed. It calculates total nutritional values (protein, fat, carbs, and calories) based on the quantity.

FoodAdd.java

This file provides functionality to add new food items to the library. It collects user input for food details and calculates nutritional values based on the quantity consumed.

Library.java

This file manages the food library. It includes methods to add, edit, delete, view, and sort food items. It also provides functionality to display the library with optional sorting based on user input.

inFileWriter.java

This file contains the logic for writing food data to a file. It formats the data into a table-like structure and saves it to a `.txt` file.

saveFile.java

This file handles saving the food library to a text file. It uses the `inFileWriter` class to write the food data into a formatted file.

ConsoleColors.java

This file provides ANSI escape codes for styling console output, including text colors, background colors, bold text, and underlined text. It is used to enhance the visual appearance of console messages.

clearConsole.java

This file contains a utility method to clear the console screen by printing ANSI escape sequences. It is used to refresh the console display.
